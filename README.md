## 什么是书籍Skill？

你有没有这种体验：书买了，读了，两周后脑子里什么都剩不下？

书籍Skill解决这个问题。它把书里的方法论提炼成一个AI工具，安装后直接在Claude里用——遇到问题，用书里的框架帮你分析，比自己翻书快10倍。

## 如何安装

1. 点击上方链接下载 `.skill` 文件
2. 在 Claude 中打开「技能设置」
3. 点击「安装本地Skill」，选择文件
4. 完成，直接对话使用
